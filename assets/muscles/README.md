# Иконки групп мышц (от дизайн-окна, 05.07.2026)

6 SVG для категорий каталога: `chest` (грудь), `back` (спина), `legs` (ноги), `shoulders` (плечи), `arms` (руки), `abs` (пресс)

## Формат

- viewBox 24×24, контурные, `fill="none"`, `stroke="currentColor"`, stroke-width 1.8, round caps - тот же стиль, что иконки таб-бара на борде
- Цвет наследуется от CSS `color` родителя: инлайнить в разметку (рекомендуется) или подключать через CSS-mask; через `<img>` currentColor работать НЕ будет
- Стилизация по дизайн-спеке (v7 в specs/2026-07-03-fit-tracker-design.md): в фильтр-чипах пикера иконка 16-18px цветом muted, у активного чипа - чип заливается лаймом, иконка тёмная (--accent-ink); в тегах мышц шторки упражнения основная мышца - лаймовый тинт, второстепенные - muted

Никакого анатомического натурализма - абстрактные глифы, как просил хендофф
